# Aviation Safety Survey Form

This is a functional survey form built as part of the FreeCodeCamp (FCC) Responsive Web Design certification. The project focuses on implementing correct HTML5 semantic tags and form syntax within an aviation industry context.

## Project Goal
The objective was to create a structured form to collect safety-related data from aviation personnel, ensuring all inputs meet the technical requirements of the FCC user stories.

## Features
- **Semantic HTML5:** Proper use of `<form>`, `<fieldset>`, `<label>`, and input types.
- **Aviation Context:** Tailored for safety reporting, including job positions, work shifts, and documentation checklists.
- **Form Validation:** Use of required attributes, type-specific inputs (email, number), and placeholders.

## Technical Requirements Met
- Linked with a unique `#title` and `#description`.
- Implemented `#survey-form` with appropriate input IDs.
- Included specialized fields:
  - **Dropdown:** For selecting professional roles.
  - **Radio Buttons:** For work shift selection.
  - **Checkboxes:** For safety documentation verification.
  - **Textarea:** For detailed safety suggestions.

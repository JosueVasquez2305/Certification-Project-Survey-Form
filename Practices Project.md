📝 Survey Form - Aircraft Maintenance Safety Survey (Structure Practice)
This project focuses on the core structure of a web form using pure HTML5. The goal was to master form elements, data types, and attributes without external styling.

🛠️ Technologies Used
HTML5 Only: Focused on semantic structure and input handling.

💡 Learning Objectives
In this project, I focused on:

Form Elements: Implementing radio buttons, checkboxes, dropdown menus, and text areas.

Input Attributes: Using required, placeholder, and specific type attributes (email, number) to handle data correctly.

Labels & IDs: Correcting the association between <label> and <input> for better accessibility.

Form Hierarchy: Organizing the fields using logical flow.

Note: This is a "skeleton" project to master HTML fundamentals. I plan to add CSS styling in a future update.
